About Me

Web Developer & Graphic Designer

Halo semua, nama saya Yusria Nurhasanah. Saya berasal dari Jambi dan saat ini sedang menempuh pendidikan sebagai mahasiswa Teknik Informatika di Universitas Pasundan (UNPAS).

Latar belakang saya berasal dari SMK Negeri 1 jurusan multimedia. Selama sekolah, saya banyak belajar editing, desain visual, dan perfilman menggunakan aplikasi Adobe.

Saya juga pernah magang di stasiun TV di Jambi dan belajar langsung proses produksi video dari pengambilan gambar hingga editing dalam perfilman.

Saat ini saya fokus mengembangkan HTML dan CSS untuk menggabungkan desain visual dan web development menjadi karya digital yang profesional.
